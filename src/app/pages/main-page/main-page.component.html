Maintain User Security Assignments

This application maintains security groups, and allows assigments of users and applications to those groups, for a fictional application security system. Below is the database schema that this application maintains. The data is maintained using JPA for database access, and Jackson/Spring for REST services. The postgres database includes constraints to demonstrate error handling.
